# 🚀 AWS Static Website Hosting using Amazon S3

## 📖 Overview
This project demonstrates how to host a static website using Amazon S3.

## 🏗️ Architecture
User → Amazon S3 Bucket → Static Website Hosting

## ☁️ AWS Services Used
- Amazon S3

## ✨ Features
- Static website hosting
- Public website access
- Bucket policy configuration
- Static Website Hosting enabled

## 🛠️ Steps Performed
1. Created an Amazon S3 bucket.
2. Uploaded the website files.
3. Enabled Static Website Hosting.
4. Configured bucket permissions.
5. Made the website publicly accessible.
6. Verified the deployment.
